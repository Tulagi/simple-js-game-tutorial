# Simple JavaScript Game Tutorial

这是一个简单的 JavaScript 游戏开发教程项目。

## 项目介绍

本教程将带你从零开始创建一个简单的打砖块游戏。

## 功能特性

- 基础的游戏循环
- 键盘控制
- 碰撞检测
- 分数系统

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/Tulagi/simple-js-game-tutorial.git
```

2. 打开 `index.html` 文件

3. 开始游戏！
